# HawkEye — Scouting System
**FRC Team 3310 Black Hawks · REBUILT 2026**
> STATE CHAMPS EDITION 

## Live App
**[Launch HawkEye →](https://brycecustard.github.io/HawkEye-Beta/)**

## Features
- **Match Scouting** — Auton & teleop fuel tracking, climb levels, field navigation, robot status, fouls, and comments
- **Pit Scouting** — Drivetrain, archetype, battery count, hopper capacity, and robot notes
- **QR Code Export** — Scan-to-transfer data pipeline for rapid collection
- **TBA Integration** — Auto-fills team name and location from The Blue Alliance
- **Offline Support** — Works without WiFi after first load (PWA with service worker caching)
- **Shooter Speed Rating** — Live balls-per-second rate calculator with tier classification
- **Saved Data Management** — View, export, and clear scouted entries
- **Mobile-First** — Designed for phone use at competition with touch-optimized controls

## Install on Your Phone
1. Open the live link above on your phone (Safari on iPhone, Chrome on Android)
2. Tap **Share** → **Add to Home Screen**
3. HawkEye will appear as an app icon — works offline after first load
